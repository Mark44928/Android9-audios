# Contributing to Android 9 Audios

Thanks for your interest in contributing!

## How to Contribute

1. **Fork** the repository
2. **Create** a feature branch (`git checkout -b feature/my-feature`)
3. **Commit** your changes (`git commit -m "Add my feature"`)
4. **Push** to the branch (`git push origin feature/my-feature`)
5. **Open** a Pull Request

## Ideas for Contributions

- Add audio files from other Android versions
- Fix naming inconsistencies
- Add metadata (bitrate, duration)
- Create a better archive format

## Guidelines

- Use consistent naming conventions
- Verify audio files are not corrupted
- Include the source Android version
